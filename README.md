## 简介

日常助手 Pro 是一款轻量级的 H5 效率管理应用，集成了待办事项、备忘录、习惯打卡、番茄钟计时器和 AI 智能助手等功能，帮助你高效管理日常任务和提升专注力。

## 功能特性

- ✅ **待办事项**：支持优先级设置（高/中/低）、截止日期、完成状态标记
- 📝 **备忘录**：支持分类标签（工作/生活/学习/灵感/其他）、置顶、富文本编辑
- 🎯 **习惯打卡**：自定义习惯、每日打卡、连续天数统计、完成率可视化
- ⏱️ **番茄钟**：专注计时、自定义时长、休息提醒
- 🤖 **AI 智能助手**：可配置 OpenAI 兼容 API，内置对话界面
- 🌓 **深色模式**：支持亮色/暗色主题切换
- 📱 **响应式设计**：适配手机、平板、桌面端
- ☁️ **数据持久化**：基于 localStorage 的本地数据存储
- 🔐 **用户系统**：支持注册/登录，数据隔离
- 📜 **每日箴言**：首页展示每日一句励志语录

## 本地开发

### 方式一：直接打开

直接用浏览器打开 `index.html` 即可使用。

```bash

```

### 方式二：本地服务器

推荐使用本地服务器以获得更好的体验：

```bash
# Python 3
python -m http.server 8080

# Node.js（需先安装 http-server）
npx http-server -p 8080

# VS Code Live Server 插件
```

然后访问 `http://localhost:8080`

## AI 功能配置

1. 打开应用后，点击底部「AI 助手」Tab
2. 点击模型标签（⚙️ API 未配置）进入配置
3. 输入你的 API Key 和 API 端点 URL
4. 支持任意 OpenAI 兼容接口（如 OpenAI、Azure、本地模型等）

## 使用说明

### 待办事项
- 输入任务名称，选择优先级和截止日期，点击「添加」
- 点击任务前的圆圈标记完成
- 点击删除按钮移除任务

### 备忘录
- 点击「新建备忘」创建笔记
- 支持标题、内容、分类标签
- 可通过置顶功能将重要笔记固定在顶部

### 习惯打卡
- 创建新习惯并设置目标
- 每日点击打卡记录完成状态
- 查看连续天数和完成率

### 番茄钟
- 默认 25 分钟专注 + 5 分钟休息
- 可自定义专注和休息时长
- 计时结束后自动提醒
